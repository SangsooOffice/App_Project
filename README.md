# Project Title
Development of an app that informs the expiration date of ingredients in the refrigerator. This project was conducted during the Android programming class in the first semester of 2022.

# Why I do this project?
Food ingredients asking for food ingredients are a project that informs the expiration date. This project receives food information from the user and informs the user of the food ingredients that have one day left of the expiration date in advance. The project asks for food ingredients not only gives users information about the expiration date but also the use by date

# Features of this application
- Ability to store data
- The ability to check the food information in the list
- Ability to categorize by data type
- Ability to compare expiration date with current date
- The ability to sound an alarm a day before the expiration date
- The ability to delete food information
